# Preguntas frecuentes

## ¿Sirve para cualquier programa?

Sirve para cualquier programa distribuido como archivo AppImage. No administra instaladores `.deb`, `.rpm`, Snap o Flatpak.

## ¿Necesita sudo?

No. Instala todo dentro del directorio personal del usuario.

## ¿Elimina mis proyectos?

No. `--remove` elimina el AppImage, el lanzador, el acceso del menú y el ícono administrados. No elimina documentos creados por la aplicación.
